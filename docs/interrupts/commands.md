The following interrupt-related APIs are provided by irq.h:

    IRQ_CONNECT
    IRQ_DIRECT_CONNECT
    ISR_DIRECT_HEADER
    ISR_DIRECT_FOOTER
    ISR_DIRECT_PM
    ISR_DIRECT_DECLARE
    irq_lock()
    irq_unlock()
    irq_enable()
    irq_disable()
    irq_is_enabled()

The following interrupt-related APIs are provided by kernel.h:

    k_is_in_isr()
    k_is_preempt_thread()
